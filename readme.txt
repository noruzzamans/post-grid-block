== Description ==

The Post Grid Block plugin brings a powerful and flexible post grid block to your WordPress editor, allowing you to easily display your posts in a customizable grid layout. Whether you're a beginner or an experienced developer, this block gives you the flexibility to showcase your content in an attractive and organized manner.

Key Features:

Customizable Layout: Adjust the number of columns and rows to create the perfect grid for your content.
Post Selection: Choose which posts to display based on categories, tags, or custom post types.
Sorting Options: Order your posts by date, title, or custom fields to present your content exactly how you want.
Responsive Design: Your post grid will automatically adapt to different screen sizes, ensuring an optimal display on mobile, tablet, and desktop devices.
Styling Options: Customize the appearance of your post grid to match your theme. Change colors, fonts, and other styles for a unique look.
Excerpt Control: Choose whether to display post excerpts and control their length for a clean, consistent look.
Featured Image Support: Showcase your posts' featured images in the grid for added visual appeal.
Pagination: Add pagination to your grid for easy navigation through large sets of posts.
User-Friendly Interface: Designed for the WordPress block editor (Gutenberg), the Post Grid Block plugin integrates smoothly into the editor experience, providing intuitive settings without the need for additional coding.

== Installation ==

There are no prerequisites for installing **Post Grid Block**. You can follow the steps below to complete the installation.

= Automatic Installation =
1.  Go to `Plugins > Add New` screen from your WordPress website dashboard.
2.  Search for `Post Grid Block`.
3.  Install and activate the plugin, that's it!
